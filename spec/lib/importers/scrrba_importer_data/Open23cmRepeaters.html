Southern California Open 1200MHz FM Voice Repeaters
Last Updated 7/20/2024

The following is a list of open 23cm FM voice repeaters and special use frequencies in the Southern California region.  Open repeaters are operated by individuals and clubs for the benefit of all radio amateurs.  Since they are classified as "open", no prior authorization or permission is necessary before using these repeaters.

At the specific request of each stations trustee, the following stations have been coordinated as open public use repeaters.

There is a great deal of other coordinated activity in this sub band.  This list is of open public use repeaters only.  Open public use repeaters are so coordinated at the specific request of the system coordinee/trustee.  This list does not include closed/private systems, and is NOT at all indicative of the total coordinated activity in this region.

All repeater inputs are -12MHz in this band.  See the Southern California 23cm Band Plan.

Please note that in early 2012 many repeaters (including the test pair) have been moved up the band to avoid interference to government radar.

Recent additions or changes are shown in green.
